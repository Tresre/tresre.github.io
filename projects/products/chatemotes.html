Chat Emotes


GIVE YOUR PLAYERS CHAT EMOTES WITH THIS SIMPLE SKRIPT!

Info:

Chat Emotes is a simple skript that allows you to use fun ASCII emotes in your server chat! The skript has 14 built-in emotes and also offers the ability to add your own ASCII emotes from in-game with a simple command!

This skript is currently created and developed by me, Tresre. If you have any questions, feel free to join my support discord and ask.


Setup:

By default, this skript requires no setup and works right away. However, it does offer customization options with a config inside the skript for customizing who is able to use emotes. The config is located at the top of the skript, and I recommend you don't change any code below the config section. Please keep in mind that this skript does need at least 1 plugin slots open to install skript if it isn't already installed.


Default permissions (can be changed in the config):
-Set to anyone by default, but can be changed so that only specific people can use emotes.


Required addons:
-None



-In Game Commands:

The skript has some in-game commands that you can use to control the skript! The commands are as follows:
-/emotelist - List all available emotes.
-/emotes - Add/remove custom emotes for your server.
