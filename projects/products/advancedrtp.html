Advanced RTP


ALLOW YOUR PLAYERS TO RANDOMLY TELEPORT IN A WORLD FOR FACTIONS, SMPS, AND MORE!

Info:

Advanced RTP (Advanced Random Teleport) is a simple skript that allows you to give your players the ability to teleport to a random location inside a world of your choosing! The skript will also ensure that they don't get trapped inside a cave system or get stuck in the sky, and that they arent teleported into lava or fire. The skript can be used for a factions server, an smp server, or any other server!

This skript is currently created and developed by me, Tresre. If you have any questions, feel free to join my support discord and ask.


Setup:

By default, this skript requires no setup and works right away. However, it does offer customization options with a config inside the skript for customizing the furthest teleport distance, what world to teleport in, and how long the command cooldown should be. The config is located at the top of the skript, and I recommend you don't change any code below the config section. Please keep in mind that this skript does need at least 1 plugin slot open to install skript if it isn't already installed.


Required addons:
-None


-In Game Commands:

The skript has an in-game command that you can use to control the skript:
-/rtp - Randomly teleport inside the world specified in the config.


The 5 built-in features are as follows:

-Customize the range of random teleports.
-Set what world the rtp command can be used in.
-Customize the cooldown for the rtp command.
-Player's wont get trapped underground or stuck in the sky.
-Player's wont get teleported into lava or fire.
